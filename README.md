# Student Management System

## Project Description
This is an ASP.NET Core Web API project for managing student records.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger

## How to Run
1. Clone the repository
2. Open in Visual Studio or VS Code
3. Run: dotnet run
4. Open: https://localhost:xxxx/swagger
5. Open: https://localhost:44377/intex.html
